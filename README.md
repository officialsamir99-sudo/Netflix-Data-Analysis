# Netflix Data Analysis

This project is a Netflix Data Analysis project created using Python and Jupyter Notebook. The project analyzes Netflix movies and TV shows data to find useful insights and trends.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

## Project Features

* Data Cleaning
* Exploratory Data Analysis (EDA)
* Data Visualization
* Rating Analysis
* Year-wise Content Analysis
* Movies vs TV Shows Comparison

## Visualizations

* Pie Charts
* Bar Graphs
* Line Charts

## Goal of the Project

The main goal of this project is to understand Netflix content trends and improve data analysis skills using real-world datasets.
